# Follow-up — No Response (5 days)

> Summary line 1: Gentle 5-day-later follow-up if original cold outreach got no reply.
> Summary line 2: Tone: respectful, brief, no guilt-tripping, one new artifact added.
> Summary line 3: Goal: re-trigger reply with a fresh angle, OR close the loop politely.

## Tone notes
- Inbox guilt is the #1 reason follow-ups fail. Acknowledge their bandwidth explicitly.
- Add ONE new artifact, not a re-pitch of the original. Marginal new value re-opens the thread.
- Always include a graceful exit: "no follow-up if not now".
- ≤80 words.

## Subject line variants
- A: "Re: {{original_subject}}" (same thread, keeps context)
- B: "Quick add to my last note — {{new_artifact_name}}"
- C: "Last try — happy to close the loop if not now"

## Body (peer / CTO / engineer audience)
Hi {{first_name}},

Following up on my note from {{original_date}} — inbox-aware, so this is a one-time nudge.

One new artifact since then: {{new_artifact_link}} — {{one_line_what_it_is}}. ~2 minutes.

If now isn't the right cycle for Vendor B, no follow-up — totally understood. Tarball stands on its own at {{tarball_link}}.

Best,
{{sender_name}}

## Body (CMO / CFO / CEO audience — slightly more formal)
Hi {{first_name}},

Following up on my message from {{original_date}}. I know inbox volume is brutal, so this is the only nudge.

One addition: {{new_artifact_link}}. Two minutes if you scroll. If now isn't the right phase, fully understood — no follow-up. Happy to circle back in {{revisit_quarter}}.

Best,
{{sender_name}}

## Suggested "new artifact" options
- Fresh tarball with current date (always works, shows continued progress)
- One-pager (if you didn't send it the first time)
- Most recent industry-first dossier update
- A 30-second screen recording video
- A competitor comparison table

## Placeholder reference
- {{first_name}}, {{original_subject}}, {{original_date}}, {{new_artifact_link}}, {{new_artifact_name}}, {{one_line_what_it_is}}, {{tarball_link}}, {{revisit_quarter}}, {{sender_name}}

## Send checklist
- [ ] In the same email thread as original (preserves context)
- [ ] Sent at a different time-of-day from the original
- [ ] New artifact is genuinely new (not a re-skin of original)
- [ ] Graceful exit clause present
- [ ] Sent Tuesday–Thursday, AM target-local

## What NOT to do
- Do NOT send a third follow-up. Two-and-out.
- Do NOT add urgency tactics ("offer expires Friday"). Engineers and CFOs hate that.
- Do NOT CC anyone new. Keeps the thread clean.
- Do NOT switch channels (email → LinkedIn → email) without their consent.

## If they reply
- Engaged → straight to `warm-*-intro.md` flow
- "Not now, circle back in Q3" → calendar it, no further action
- "Not interested" → reply with one-line gracious close
