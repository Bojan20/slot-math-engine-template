# iOS · Capacitor wrapper skeleton

This directory contains a **placeholder** Xcode project layout so the
repository can be inspected without macOS / Xcode installed. The real
`App.xcworkspace` is generated by `npx cap add ios` from
`mobile/capacitor.config.json`.

Files of interest:

- `App/App/Info.plist` — bundle metadata, permissions (camera /
  photos / documents), App Transport Security exceptions for the
  Capacitor `capacitor://localhost` scheme, and the `slotstudio://`
  custom URL handler.
- `App/App/AppDelegate.swift` — minimal app delegate that defers to
  Capacitor's `ApplicationDelegateProxy`.

### Build pipeline

```bash
cd ../..              # repo root
npm run studio:build  # produce web/studio/dist
cd mobile
npx cap sync ios
npx cap open ios      # opens Xcode → Product → Archive
```

Signing requires an Apple Developer account + a provisioning profile
for `com.vanvinkl.slotstudio`. The build itself just bundles the web
output — no Swift code beyond the Capacitor scaffold.
