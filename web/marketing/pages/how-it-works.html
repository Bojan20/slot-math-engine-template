How it works

Four ideas that make the difference between a 14-day cycle and a 14-week one.

Architecture, in one diagram

  ┌──────────────────┐     ┌──────────────────┐
  │  Game designer   │ →   │   IR (YAML)      │
  └──────────────────┘     └────────┬─────────┘
                                    │
                          ┌─────────┴──────────┐
                          ↓                    ↓
              ┌────────────────────┐  ┌────────────────┐
              │ Closed-form solver │  │ Monte Carlo    │
              │   (RTP / var / hits)│  │  (≥ 1M spins)  │
              └─────────┬──────────┘  └────────┬───────┘
                        │  reconcile  ←────────┘
                        ↓
              ┌────────────────────┐
              │  Cert dossier      │ → BMM / GLI / eCOGRA / NMi
              │  (Ed25519-signed)  │
              └────────────────────┘
①

Closed-form first, MC second

Every solver computes the analytic RTP, variance, and hit distribution in < 200 ms — then validates against ≥ 1M Monte Carlo spins. If they disagree by more than the configured tolerance, the build fails. No "let it run overnight" math.

②

IR config-as-data

The game's math lives in a single YAML / JSON file. No code. Reels, paytable, wilds, bonus triggers — all declarative. Versioned in git like infrastructure. Reproducible across machines, OSes, languages.

③

Deterministic replay

Every spin is hash-chain audited. Given a seed + IR + spin index, the result is byte-identical on TS or Rust, on macOS or ARM Linux. Regulators can replay any session offline from the manifest.

④

Cert paper trail

One IR file → four lab-formatted dossiers. BMM, GLI, eCOGRA, NMi submissions auto-generated. Each manifest carries an Ed25519 signature + Merkle PAR commitment. Tamper-evident end to end.

A minimal IR file

This is a full working game: 5 reels, 3 rows, 9-symbol set, wild substitution. Engine ships a closed-form solver for it in < 1 sec. 

game:
  name: "Demo Slot"
  rtp_target: 0.965
  reels: 5
  rows: 3
  paylines: 20

symbols:
  - { id: HIGH_1, hit_rate: 0.04 }
  - { id: HIGH_2, hit_rate: 0.07 }
  - { id: HIGH_3, hit_rate: 0.10 }
  - { id: LOW_1,  hit_rate: 0.14 }
  - { id: LOW_2,  hit_rate: 0.16 }
  - { id: WILD,   substitutes: [ HIGH_1, HIGH_2, HIGH_3, LOW_1, LOW_2 ] }
  - { id: SCATTER, triggers_freespins: 10 }

paytable:
  HIGH_1: { 5: 250, 4: 100, 3: 40 }
  HIGH_2: { 5: 150, 4: 60,  3: 25 }
  LOW_1:  { 5: 50,  4: 20,  3: 10 }

freespins:
  trigger: SCATTER
  count_trigger: 3
  awarded: 10
  retrigger: true
