CORTI 200.2 · Audio cue directory

Place 1-second .mp3 sample cues here using the exact filenames below.
If a file is missing the runtime falls back to a Web-Audio synth tone
generated by `createAudioEngine().preloadDefaults()` so the studio
never throws — it just uses a placeholder sine until you drop real
samples in.

Filenames (all .mp3, ≤200 KB each):

  reel-spin.mp3     — looped while reels are spinning
  reel-stop.mp3     — fired per-reel as each reel snaps
  win-small.mp3     — 3-of-a-kind win cue
  win-big.mp3       — 5-of-a-kind / mult > 100× cue
  fs-intro.mp3      — free-spin intro fanfare
  hw-orb-land.mp3   — Hold & Win orb-land impact

Designers can also upload custom cues at runtime from the Design
persona "Audio cues" rail card; uploads override the file-system
samples per session.
