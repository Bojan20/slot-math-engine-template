## Methodology

Per fixture: 5-seed × 50K-spin Monte Carlo run via `runIRSimulation`.

Three checks per pattern:
1. **Sanity** — every seed produces finite RTP, no NaN, no crash
2. **Stability** — cross-seed relative σ ≤ 10% (proves engine convergence)
3. **Mechanic** — IR contains the declared pattern-specific feature/evaluator

A pattern PASSES if all 3 checks PASS for every fixture in its set.
